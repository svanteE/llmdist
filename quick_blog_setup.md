# Quick GitHub Pages Blog Setup

## Option 1: GitHub Pages (5 minutes)
1. Go to: https://github.com/new
2. Repository name: `[your-username].github.io`
3. Make it public
4. Upload your blog_post_r_bloggers.html as `index.html`
5. Your blog URL: `https://[your-username].github.io`
6. Submit this URL to R-bloggers via their web form

## Option 2: WordPress (10minutes) 
1. Go to: https://wordpress.com
2. Create free account
3. Create new post with your blog content
4. Copy permalink
5. Submit to R-bloggers

## Then submit via:
https://www.r-bloggers.com/add-your-blog/
