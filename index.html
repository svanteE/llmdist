🚀 llmdist Package

Fast distance computation between multivariate normal distributions

✨ Key Features

  • 50x faster than traditional methods
  • Fisher-Rao geodesic distance implementation
  • 8 different distance measures
  • Optimized C++ backend

📦 Installation

# Install from GitHub
devtools::install_github("svanteE/llmdist")
library(llmdist)

🔧 Quick Example

# Create distributions
mu1 <- c(0, 0); mu2 <- c(1, 1)
Sigma1 <- diag(2); Sigma2 <- matrix(c(2, 0.5, 0.5, 1), 2, 2)

# Compute Fisher-Rao distance
fisher_rao_distance(mu1, Sigma1, mu2, Sigma2)
# [1] 1.247449
