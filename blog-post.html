Introducing llmdist: Fast Distance Computation for R

Published: April 1, 2026 | ← Back to main page

The llmdist package provides efficient distance computation between multivariate normal distributions with up to 50x performance improvements.

🚀 Why llmdist?

Traditional distance computation in R can be slow. Our optimized C++ implementation delivers:

  • Fisher-Rao geodesic distance - first R implementation
  • 8 different distance methods in one package
  • Massive performance gains over standard approaches

📊 Benchmark Results

💡 Example Usage

library(llmdist)

# Define two distributions
mu1 <- c(0, 0)
mu2 <- c(1, 1)
Sigma1 <- matrix(c(1, 0.5, 0.5, 1), 2, 2)  
Sigma2 <- matrix(c(1, -0.3, -0.3, 1), 2, 2)

# Fisher-Rao distance
fisher_rao_distance(mu1, Sigma1, mu2, Sigma2)
# [1] 1.247449

# Compare all methods
compare_distances(mu1, Sigma1, mu2, Sigma2)
